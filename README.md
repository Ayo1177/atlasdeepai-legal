# atlasdeepai-legal

Public legal pages for **Atlas Deep AI** (Meta App Review, privacy policy links).

## Enable GitHub Pages

1. Repo → **Settings** → **Pages**
2. **Build and deployment** → Source: **Deploy from a branch**
3. Branch: **main** → Folder: **/ (root)** → Save
4. Wait 1–3 minutes for the site to publish
